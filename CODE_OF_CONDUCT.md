# Code of Conduct

## Our Commitment

HealthyPixel is committed to providing a welcoming and inclusive community for all contributors and users, regardless of background, identity, or experience level.

## Expected Behavior

We expect all members of the HealthyPixel community to:

- **Be Respectful**: Treat all others with kindness and respect
- **Be Constructive**: Provide helpful feedback, not criticism or attacks
- **Be Inclusive**: Welcome diverse perspectives and ideas
- **Be Professional**: Keep discussions on-topic and productive
- **Acknowledge Ownership**: Understand and accept the LICENSE terms regarding contribution ownership

## Unacceptable Behavior

The following behaviors are unacceptable and will result in removal:

- Harassment, discrimination, or hate speech of any kind
- Spam, trolling, or deliberate disruption
- Sharing others' private information without consent
- Aggressive or threatening language
- Discrimination based on race, gender, sexual orientation, disability, or other characteristics
- Attempts to avoid or circumvent the LICENSE terms

## Enforcement

Violations of this code of conduct will result in:

1. **First Offense**: Warning and removal of offensive content
2. **Second Offense**: Temporary ban from the repository
3. **Severe Offenses**: Immediate permanent ban

## Reporting

If you witness a violation or feel unsafe, please:

1. **Report via GitHub**: Use private issue/message to maintainers
2. **Report via Email**: Contact the creator directly
3. **Escalate if Needed**: If maintainer response is inadequate, escalate to GitHub

All reports will be handled confidentially and promptly.

## Scope

This code of conduct applies to:

- GitHub repository (issues, PRs, discussions)
- All community spaces affiliated with HealthyPixel
- Official project events or gatherings
